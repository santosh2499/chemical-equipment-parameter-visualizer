# Demo Video Script (2-3 minutes)

## Introduction (15 seconds)

"Hello! This is a demonstration of the Chemical Equipment Parameter Visualizer, a hybrid application that runs both as a web application and desktop application."

## Features Overview (15 seconds)

"The application allows users to:
- Upload CSV files containing chemical equipment data
- Visualize data through interactive charts
- Generate PDF reports
- Manage dataset history"

## Web Application Demo (60 seconds)

### 1. Registration & Login (15 seconds)
- Show registration page
- Create a new account
- Login with credentials

### 2. Upload Dataset (20 seconds)
- Navigate to Upload page
- Select sample_equipment_data.csv
- Enter dataset name
- Click Upload
- Show success message

### 3. View Visualizations (25 seconds)
- Click on uploaded dataset
- Show statistics cards
- Demonstrate interactive Chart.js charts:
  - Equipment type distribution (bar chart)
  - Type breakdown (pie chart)
  - Parameter comparison
- Scroll through equipment data table

### 4. Generate PDF Report (10 seconds)
- Click "Download PDF Report"
- Show downloaded PDF file
- Briefly open PDF to show contents

## Desktop Application Demo (60 seconds)

### 1. Login (10 seconds)
- Launch desktop application
- Login with same credentials

### 2. Dashboard (10 seconds)
- Show dataset list
- Display dataset statistics

### 3. Upload New Dataset (15 seconds)
- Go to Upload tab
- Browse for CSV file
- Enter dataset name
- Upload successfully

### 4. Visualizations (25 seconds)
- Select dataset from list
- Show Matplotlib charts:
  - Pie chart for type distribution
  - Multi-bar chart for parameters
- Demonstrate smooth navigation

### 5. Download Report (10 seconds)
- Click "Download PDF Report"
- Save file
- Show saved location

## Backend Features (15 seconds)

- Show Django admin panel
- Display database with datasets
- Show last 5 datasets limit feature

## Conclusion (15 seconds)

"This hybrid application demonstrates:
- Common Django REST API backend
- React web frontend with Chart.js
- PyQt5 desktop frontend with Matplotlib
- User authentication
- Data visualization
- PDF report generation
- Dataset history management

Thank you for watching!"

## Recording Tips

1. **Preparation**:
   - Clear browser cache and history
   - Close unnecessary applications
   - Prepare sample CSV file
   - Test all features beforehand

2. **Recording**:
   - Use screen recording software (OBS Studio, Camtasia, etc.)
   - Record at 1080p resolution
   - Enable microphone for narration
   - Keep mouse movements smooth

3. **Editing**:
   - Add transitions between sections
   - Include text overlays for key features
   - Add background music (optional)
   - Export as MP4

4. **Upload**:
   - Upload to YouTube/Google Drive
   - Set appropriate privacy settings
   - Add to README.md
